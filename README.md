## Sound service for abcdesktop.io for kubernetes

```
git clone -b 3.2 git://github.com/abcdesktopio/oc.pulseaudio
cd oc.pulseaudio
```



### build `oc.pulseaudio` image

```
docker build -t abcdesktopio/oc.pulseaudio:3.2 .
```
